Projects

TxT (Google Play Store)

TxT is an independent real-time chatting application that allows you to write anything you want in your native language, and the person you are chatting with will read the exact same conversation but in their
native language.

For this project I used NodeJS for the backend along with dependencies such as Express, Socket.io, mongoose and more. This is by far the biggest project I have ever worked on/still working on.

Things I learned:

  • How real-time chatting applications work
  • How to write clean and scalable code
  • How to design APIs better
  • Database design

PocketWeather (GitHub)

PocketWeather is a free and open source minimalistic weather app. It is designed to show you the most important data about the weather without too much clutter on the screen.

I decided to work on this project because I wanted to learn Flutter and Dart plus I always wanted to have a minimalistic weather app on my phone.

This project was developed with Flutter/Dart, NodeJS and Go. It has two versions of the same backend: One made with NodeJS and the other one with Go

Things I learned:

  • Basics of how Flutter and Dart work
  • A bit of Android development in general

Vidy (GitHub)

Vidy is an opensource, lighweight, minimalistic and efficient alternative to Plex

I wanted to work on this project because I wanted to make a Plex alternative that was more suitable for me, because Plex had too many features and was resource hungry.

I worked on this project to enhance my Go skills.

Things I learned:

  • http library in Go
  • Using templates
  • Serving files from a server to a client

node-youtube-listener (GitHub)

node-youtube-listener is a script made with NodeJS that keeps checking if a specified YouTube channel has uploaded a video and downloads it automatically.

I like hoarding music, downloading music manually from multiple channels was a bit of a pain, so I decided to make something that will automate that.

Things I learned:

  • How web crawlers work
  • Async/Await in JavaScript
  • Reading and writing files with NodeJS
